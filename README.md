# Weaver Tutorial

This project is a support for the Weaver tutorial.

## Installation

Run these commands in a terminal:

```bash
$ git clone git@github.com:trupin/WeaverTutorial.git
$ cd WeaverTutorial
$ open Sample.xcodeproj
```
